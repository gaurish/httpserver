# Httpserver

Start a http server in any directory

## Installation

Add this line to your application's Gemfile:

    gem 'httpserver'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install httpserver

## Usage

go to any directory

`httpserver .`

will start a webserver in that directory
